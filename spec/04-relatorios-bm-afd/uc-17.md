UC-17: Consultar Funil de Faturas
Utilizadores: Coordenador (principal), Admin, Procurement
Prioridade: Alta (Should Have / Nice to Have no MVP)
Objetivo:
Fornecer ao Coordenador uma visão clara e visual do fluxo de processamento das faturas através de um funil horizontal, permitindo identificar rapidamente estrangulamentos no processo e monitorizar a quantidade e valor das faturas em cada etapa do workflow.


Fluxo Principal
1. O utilizador (Coordenador) acede ao Dashboard Estratégico.
2. O sistema carrega automaticamente o Funil de Faturas na secção de visualizações gráficas.
3. O dashboard apresenta um gráfico de barras horizontal mostrando:
  3.1 Cada etapa do workflow
  3.2 Quantidade de faturas em cada estado
  3.3 Valor total das faturas em cada estado
4. O utilizador pode visualizar os dados por:
  4.1 Quantidade de faturas
  4.2 Valor monetário (em AOA)
5. Ao clicar em qualquer barra do funil, o sistema abre uma lista detalhada das faturas correspondentes a esse estado, com possibilidade de filtro e exportação.
6. O funil atualiza-se automaticamente (refresh a cada 5 minutos ou em tempo real).


Critérios de Aceitação
Item,Detalhe,Obrigatório
Tipo de gráfico,Gráfico de barras horizontal,Sim
Estados apresentados,"Triagem, Em Pareceres, Procurement, Finanças (e outros relevantes)",Sim
Dupla métrica,Quantidade de faturas + Valor Total (AOA),Sim
Interatividade,Click em cada barra → lista detalhada das faturas,Sim
Atualização,Atualização automática periódica,Sim


Estados do Funil (Recomendado)
1. Triagem
2. Em Pareceres (ou subdividido por tipo de parecer)
3. Pareceres Concluídos
4. Em Validação Procurement
5. Em Validação Financeira
6. Paga / Arquivada (opcional, para comparação)



Requisitos de Visualização
1. Gráfico de Barras Horizontal:
  1.1 Eixo Y: Nome dos estados
  1.2 Eixo X: Quantidade ou Valor
  1.3 Cada barra deve mostrar:
    1.3.1 Número de faturas
    1.3.2 Valor total formatado (ex: 12.450.000 AOA)
    1.3.3 Percentagem do total (opcional)
2. Cores por estado (recomendado):
  2.1 Triagem → Azul claro
  2.2 Em Pareceres → Laranja
  2.3 Procurement → Roxo
  2.4 Finanças → Verde escuro
  2.5 Estagnadas → Vermelho
3. Drill-down: Clique na barra → abre tabela com lista de faturas desse estado, incluindo número da fatura, contrato, entidade, valor, responsável e tempo no estado.


Requisitos Não Funcionais
1. Desempenho: Carregamento rápido (< 2 segundos)
2. Responsividade: O gráfico deve adaptar-se bem a tablet e telemóvel
3. Atualização: Dados atualizados automaticamente (websocket ou polling)
4. Exportação: Possibilidade de exportar o funil ou a lista detalhada em Excel/PDF
5. Acessibilidade: Texto alternativo para leitores de ecrã


Integrações Fortes
1. UC-06 a UC-10 (Workflow completo de faturas)
2. UC-12: Alerta de Estagnação (48h)
3. UC-16: Visualizar KPIs Globais
4. UC-05: Cálculo de Saldo Real


Nota importante:
O Funil de Faturas é uma das visualizações mais importantes mencionadas no caderno de encargos do Módulo 5 – Dashboard de Gestão Estratégica. Ajuda o Coordenador a identificar rapidamente onde estão os gargalos no processo de aprovação e pagamento.
